``TracerData``
======================
..
	(Class)

:doc:`Click here to see the source code <tracer_data_hh_src>`

**Brief Description:** This class is used to store tracing data for functions.

**Detailed Description:**

    It keeps track of the number of calls and the total execution time.
    By using a linked list, it allows for easy accumulation of tracing
    data across multiple function calls.

*In file* ``src/understanding/tracer_data.hh``
